Nice work! The links in the navigation bar are styled.

Next, let's style the large feature section. In index.html, the <div class="jumbotron">..</div> groups elements that are part of the large feature section.

Instructions

In main.css

  1. Make a new CSS rule that uses the class selector .jumbotron
  2. Set the height to 500px
  3. Set the background-image to url('https://goo.gl/04j7Nn')
?
Stuck? Get a hint! Hint

Your CSS rule should look something like this:

.jumbotron {
  background-image: url('https://goo.gl/04j7Nn');
  height: 500px;
}
